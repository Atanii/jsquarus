## Squarus clone in HTML, CSS, JavaScript (JQuery)

## Original game: https://saucetin.itch.io/squarus

------

### Play:

You can try and play this game here: https://atanii.github.io/jsquarus/

### Controls:

| Key  | command                       |
| ---- | ----------------------------- |
| r    | reload level and loose scores |
| w    | up                            |
| d    | right                         |
| s    | down                          |
| a    | left                          |
